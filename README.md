# Railway Reservation System

A full-stack oriented project to simulate a railway ticket booking system using core **Data Structures and Algorithms (DSA)**.

The project will implement seat allocation, waiting list management, and priority bookings using in-memory data structures, followed by backend and frontend integration.

---

## Project Status
Planning and initial setup

---

## Objective
To demonstrate real-world application of DSA concepts in a scalable ticket booking system.
